## To run it locally:

1. go to folder Project2ECommerce
   nvm use 14
   npm install

2. cd frontend
   nvm use 14
   npm install

3. go to folder Project2ECommerce
   add you own .env, it should be the same file level as package.json
   
4. npm run dev
